# 🔷 4. Indicator Framework (หัวใจ Dashboard)

## 4.1 Health Need Index

```
HNI = weighted(population + elderly + chronic + risk)
```

## 4.2 Workforce Capacity Index (WCI)

```
WCI = (doctor*5 + nurse*3 + other*1) / population
```

---

## 4.3 Gap Index (GI)

🔥 สำคัญที่สุด

```
GI = HNI - WCI
```

---

## 4.4 Priority Score

```
Priority = GI * SGI * (1 - HSI)
```

👉 ใช้จัดลำดับ "ควรแก้ที่ไหนก่อน"
